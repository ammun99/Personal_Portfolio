# Personal_Portfolio
Designed and built a fully interactive portfolio using Vue 3, Tailwind CSS, and GSAP animations. Implements component-driven architecture, scroll-triggered animations, and responsive UI patterns.
# 🌐 Vue Portfolio — Amrutha Nagaraj

A modern, animated, and responsive personal portfolio built with **Vue 3**, **Vite**, **Tailwind CSS**, and **GSAP**.  
Designed to showcase frontend engineering skills, clean UI design, and smooth user interactions.

🔗 **Live Demo:** https://ammun99.github.io/Portfolio/  
📂 **Source Code:** https://github.com/ammun99/Portfolio

---

## ✨ Features

- ⚡ Built with **Vue 3 + Vite** for fast development and performance
- 🎨 **Tailwind CSS** for modern, responsive UI
- 🎬 **GSAP animations** (hero entrance, scroll-based animations)
- 🎥 **Subtle video background** in Hero section (blurred + dark overlay)
- 🧭 Sticky **Navbar with scroll progress indicator**
- 🗂️ **Projects section** with animated cards
- 🎯 **Skills filter** using Vue state + transitions
- ✉️ Interactive **Contact form UI**
- 📱 Fully **responsive** (desktop & mobile)
- ♿ Focused on readability and clean layering

---

## 🛠️ Tech Stack

- **Frontend:** Vue 3 (Composition API), JavaScript
- **Styling:** Tailwind CSS
- **Animations:** GSAP
- **Build Tool:** Vite
- **Deployment:** GitHub Pages / Vercel (optional)
